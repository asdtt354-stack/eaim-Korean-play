시작 전 준비

Gemini API 키 설정 · 최초 1회
🔑 API 키가 필요한 이유

AI 시 추천, AI 글감 추천, AI 첨삭·수정·이어쓰기, AI 음악 추천 기능 모두 Google Gemini 2.5 Flash를 사용합니다. API 키 없이도 기본 활동(감정 선택, 글쓰기)은 가능하지만 AI 기능은 키가 필요합니다.

💡

무료로 충분해요!

Google AI Studio 무료 티어 — 분당 15회, 하루 1,500회 무료. 수업 활용에 충분합니다.

📋 API 키 발급 4단계

Google AI Studio 접속

Chrome에서 아래 주소로 이동 후 Google 계정으로 로그인

https://aistudio.google.com

API 키 생성

왼쪽 메뉴 Get API key → Create API key 클릭

키 복사

AIzaSy...로 시작하는 키를 복사해 안전하게 보관

앱에 입력

각 앱 하단 "선생님이신가요?" → 교사 패널 → Gemini API 키 입력 → 저장. 브라우저에 기억됩니다.

⚠️

주의사항

API 키는 개인 정보입니다. SNS·공개 게시물에 올리지 마세요. 학생 기기에는 선생님이 직접 입력해 주세요.

📜 시 감상 & 창작

eaim-korean-poetry.html · 7단계
🚀 전체 흐름
💭

Step 1 — 감정/주제 입력 시작

지금 느끼는 감정이나 주제를 입력하거나, 12개 감정 칩 중 하나를 클릭해요. 외로움·그리움·설렘·희망 등 다양하게 선택할 수 있어요.

📚

Step 2 — AI 시 추천 AI

Gemini AI가 감정에 어울리는 한국 시 3편(윤동주·김소월·백석 등)을 추천해줘요. 추천 이유도 함께 표시돼요. 마음에 드는 시를 클릭해 선택하세요.

🎨

Step 3 — 감정 탭 선택 감상

시 전문을 읽으며 느낀 감정을 15개 탭 중 여러 개 클릭해요. 슬픔·그리움·감동·쓸쓸함·몽환적 등 다양한 감정어가 준비돼 있어요.

✏️

Step 4 — 감상문 쓰기 글쓰기

왜 그런 감정이 들었는지, 어떤 구절이 마음에 남았는지 자유롭게 써요. AI 첨삭 버튼으로 피드백을 받을 수 있어요.

🌸

Step 5 — 나만의 시 창작 창작

읽은 시에서 영감을 받아 나만의 시를 써봐요. AI 첨삭·AI 영감받기 버튼을 활용할 수 있어요.

📖

Step 6 — 감상 일지 완성 완성

감상한 시·느낀 감정·감상문·나만의 시가 한 페이지에 모여요. 인쇄하거나 텍스트 파일로 저장할 수 있어요.

🎵

Step 7 — 음악 & 낭독 낭독

배경음악을 선택하고(유튜브/파일업로드/AI추천) TTS로 시를 낭독해요. 속도 조절 가능, 감상 시와 나의 시 모두 낭독 가능해요.

💡

장르별 개요 자동 생성

장르를 선택하면 Step 3 개요 구조가 자동으로 바뀌어요. 일기는 처음·중간·끝, 주장하는 글은 주장·근거1·근거2·마무리 형태로요.

🚀 전체 흐름
📝

Step 1 — 장르·주제·감정

6가지 장르 중 하나를 선택하고, 쓰고 싶은 주제를 입력해요. 감정 칩 12개로 분위기도 설정할 수 있어요.

💡

Step 2 — AI 글감 추천

Gemini AI가 장르와 주제에 맞는 구체적인 글감 3가지를 추천해줘요. 마음에 드는 글감을 선택하세요.

🗂️

Step 3 — 개요 짜기

장르에 맞는 개요 구조에 간단히 메모해요. AI 개요 도움받기 버튼으로 AI의 제안을 참고할 수 있어요.

✍️

Step 4 — 본문 쓰기 + AI 도구

자유롭게 본문을 써요. AI 도구 4종(수정·첨삭·이어쓰기·다음 내용 제안)을 필요할 때 선택해서 사용할 수 있어요.

🌟

Step 5 — AI 최종 다듬기

전체 글을 AI가 종합 첨삭하거나, AI가 전체를 자연스럽게 다듬어줘요. 적용 여부는 학생이 직접 선택해요.

📖

Step 6 — 완성 & 저장

글감·개요·본문이 한 페이지에 완성돼요. 인쇄하거나 텍스트 파일로 저장해요.

🎵

Step 7 — 음악 & 낭독

배경음악과 함께 완성된 글을 TTS로 낭독해요.

🎵 음악 & 낭독

Step 7 · 두 앱 동일
🎶 배경음악 3가지 방식

🎬 유튜브 아이프레임

  • 유튜브 URL 붙여넣기
  • 일반·단축 링크 모두 OK
  • 앱 안에서 바로 재생
  • 학교 유튜브 허용 환경 필요

📁 파일 업로드

  • mp3·wav·ogg 지원
  • 드래그앤드롭 가능
  • 인터넷 없이도 재생
  • 저작권 없는 음원 사용 권장

🤖 AI 음악 추천

시/글의 분위기를 분석해서 낭독에 어울리는 음악 4곡을 추천해요. 카드 클릭 → 검색어 복사 → 유튜브에서 검색 → URL 복사 → 유튜브 탭에 붙여넣기

⚠️

유튜브 재생이 안 될 때

영상이 퍼가기 차단된 경우예요. 유튜브에서 공유 → 퍼가기 옵션이 있는 영상으로 교체하거나, 파일 업로드 방식을 사용하세요. 클래식·국악 영상은 대부분 퍼가기 허용돼 있어요.

❓ 자주 묻는 질문

FAQ
AI 기능이 작동하지 않아요.
교사 패널에서 Gemini API 키가 올바르게 입력됐는지 확인하세요. AIzaSy...로 시작하는지, 앞뒤 공백이 없는지 확인해 주세요.
유튜브 음악이 재생되지 않아요.
두 가지 원인이 있어요. ① 학교 네트워크에서 유튜브가 차단된 경우 → 파일 업로드 방식 사용. ② 영상의 퍼가기가 차단된 경우 → 다른 영상(클래식·국악)으로 교체하세요.
AI가 추천한 시가 없거나 이상해요.
감정/주제 입력이 너무 짧거나 모호할 때 발생해요. "외로움", "가을 방과 후" 처럼 구체적으로 입력하면 더 좋은 결과가 나와요. 다시 시도 버튼을 눌러보세요.
학생 활동 기록이 다른 기기에서 안 보여요.
현재 버전은 localStorage 기반으로 같은 기기·브라우저 내에서만 공유됩니다. 교사는 CSV 내보내기로 데이터를 통합 관리하세요.
TTS 소리가 나오지 않아요.
Chrome 브라우저 사용 여부를 확인하세요. 첫 번째 사용자 클릭 이후에만 소리가 활성화되는 경우가 있어요. 기기 볼륨·음소거도 확인해 주세요.
AI 수정 기능을 쓰면 원래 글이 사라지나요?
아니요! AI 수정 결과가 별도 박스로 표시되고, 적용하기 버튼을 눌러야만 바뀌어요. 취소 버튼을 누르면 원래 글이 유지됩니다.
Gemini API 사용 비용이 발생하나요?
Google AI Studio 무료 티어는 분당 15회, 하루 1,500회 요청이 무료예요. 일반 수업 활용 범위에서는 비용이 발생하지 않습니다.
