시작 전 준비

Gemini API 키 설정 · 반/모둠 만들기 · 최초 1회
🔑 API 키가 필요한 이유

AI 시 추천, AI 글감 추천, AI 첨삭·수정·이어쓰기, AI 기사 생성, AI 퀴즈, AI 음악 추천 기능 모두 Google Gemini 2.5 Flash를 사용합니다. API 키 없이도 기본 활동(감정 선택, 글쓰기, 직접 입력)은 가능하지만 AI 기능은 키가 필요합니다.

💡

무료로 충분해요!

Google AI Studio 무료 티어 — 분당 15회, 하루 1,500회 무료. 수업 활용에 충분합니다.

📋 API 키 발급 4단계

Google AI Studio 접속

Chrome에서 아래 주소로 이동 후 Google 계정으로 로그인

https://aistudio.google.com

API 키 생성

왼쪽 메뉴 Get API key → Create API key 클릭

키 복사

AIzaSy...로 시작하는 키를 복사해 안전하게 보관

앱에 입력 (한 번만!)

4개 앱 중 아무 곳에서나 한 번만 "선생님이신가요?" → 교사 패널 → Gemini API 키 입력 → 저장. 나머지 3개 앱에도 자동으로 반영돼요.

⚠️

주의사항

API 키는 개인 정보입니다. SNS·공개 게시물에 올리지 마세요. 학생 기기에는 선생님이 직접 입력해 주세요.

🏷️ 반/모둠 이름 만들기

학생 로그인 화면의 반/모둠 선택은 고정 목록이 아니라 선생님이 직접 만드는 이름이에요. "다詩", "3반 시창작반"처럼 원하는 이름을 자유롭게 만들 수 있어요. 교사 패널의 "🏷️ 반/모둠 이름 관리"에서 추가·삭제하면, 4개 앱 학생 로그인 화면에 똑같이 나타나요.

✅

여러 기기에서 자동으로 공유돼요

4개 앱은 같은 Firebase 프로젝트(eaim-classroom)에 연결되어 있어요. 선생님이 컴퓨터에서 만든 반 이름이 학생들 각자 폰에도 바로 나타납니다.

🚀 전체 흐름
💭

Step 1 — 감정/주제 입력 시작

지금 느끼는 감정이나 주제를 입력하거나, 12개 감정 칩 중 하나를 클릭해요.

📚

Step 2 — AI 시 추천 AI

Gemini AI가 감정에 어울리는 한국 시 3편을 추천해줘요. 마음에 드는 시를 클릭해 선택하세요.

🎨

Step 3 — 감정 탭 선택 감상

시 전문을 읽으며 느낀 감정을 15개 탭 중 여러 개 클릭해요.

✏️

Step 4 — 감상문 쓰기

왜 그런 감정이 들었는지 자유롭게 써요. AI 첨삭으로 피드백을 받을 수 있어요.

🌸

Step 5 — 나만의 시 창작

읽은 시에서 영감을 받아 나만의 시를 써봐요. AI 첨삭·영감받기를 활용할 수 있어요.

📖

Step 6 — 감상 일지 완성

모든 활동이 한 페이지에 모여요. 인쇄하거나 텍스트로 저장할 수 있고, "📰 신문으로 보내기"로 나만의 시를 신문 앱 기사 재료로 보낼 수도 있어요.

🎵

Step 7 — 음악 & 낭독

배경음악을 선택하고 TTS로 시를 낭독해요.

📰 국어 신문 만들기

eaim-korean-news.html · 5단계
🚀 전체 흐름
📰

Step 1 — 신문 설정

신문 이름(AI 추천 가능)·발행일·호수를 정하고, 4가지 스타일(조선일보형·2단 컬러형·복고형·타블로이드형) 중 골라요.

✍️

Step 2 — 기사 작성

기사 추가 방식이 3가지예요: "🤖 AI 자동 생성"(주제·섹션·유형·길이 지정), "✏️ 직접 입력(모둠)", "📥 가져오기"(시·글쓰기 앱에서 보낸 내 글 불러오기). 섞어서 기사를 여러 개 추가해요. 기사마다 "🎨 AI 삽화 생성"으로 흑백 신문 일러스트도 넣을 수 있어요.

🗞️

Step 3 — 완성된 신문

선택한 스타일로 신문이 자동 조판돼요. 복사·저장·인쇄 가능해요.

✍️

Step 4 — 감상문

신문을 만들며 느낀 점을 쓰고 AI 첨삭·도움받기를 활용해요.

🎸

Step 5 — 수노 프롬프트

신문 기사들을 주제로 한 노래 가사와 수노(Suno AI) 프롬프트를 생성해요.

👥

모둠 활동에 최적화

"직접 입력" 탭은 모둠원별로 기사를 나눠 쓰고 기자 이름을 남길 수 있어, 모둠 협업 신문 제작에 특히 잘 맞아요.

🔗

시·글쓰기 앱과 연계

시 감상&창작·글쓰기 창작 앱 완성 화면에서 "📰 신문으로 보내기"를 누르면, 신문 앱 "📥 가져오기" 탭에서 같은 이름으로 로그인한 학생 본인의 글만 불러올 수 있어요. 자기 시나 글을 신문 기사로 재활용할 수 있는 거예요.

시·글쓰기 앱 Step 7 · 신문 앱은 수노 프롬프트만 지원
🎶 배경음악 3가지 방식

🎬 유튜브 아이프레임

  • 유튜브 URL 붙여넣기
  • 일반·단축 링크 모두 OK
  • 앱 안에서 바로 재생

  • mp3·wav·ogg 지원
  • 드래그앤드롭 가능

🤖 AI 음악 추천

시/글의 분위기를 분석해서 어울리는 음악 4곡을 추천해요. 카드 클릭 → 검색어 복사 → 유튜브에서 검색 → URL 복사 → 유튜브 탭에 붙여넣기

⚠️

유튜브 재생이 안 될 때

영상이 퍼가기 차단된 경우예요. 다른 영상으로 교체하거나 파일 업로드 방식을 사용하세요.

🏷️ 반/모둠 이름 관리

교사 패널의 "반/모둠 이름 관리"에서 원하는 이름(예: "다詩", "3반 신문반")을 추가하면, 학생 로그인 화면 선택지에 바로 나타나요. 삭제도 가능해요. 4개 앱이 같은 목록을 공유하므로, 한 앱에서 만든 이름이 나머지 앱에도 똑같이 보여요.

시·글쓰기·신문·게임 앱은 같은 Firebase 프로젝트(eaim-classroom)를 사용해서, 반/모둠 목록과 "신문으로 보내기"한 글이 학생 기기가 달라도 실시간으로 공유돼요. API 키는 한 곳에서만 설정하면 됩니다. 단, 각 앱의 학생 활동 기록(누가 어디까지 썼는지)은 기기별로 따로 저장돼요.

FAQ
AI 기능이 작동하지 않아요.
교사 패널에서 Gemini API 키가 올바르게 입력됐는지 확인하세요. AIzaSy...로 시작하는지, 앞뒤 공백이 없는지 확인해 주세요. 4개 앱 중 하나에만 입력하면 나머지에도 자동 반영돼요.
학생 로그인 화면에 반/모둠 목록이 안 보여요.
아직 선생님이 반/모둠 이름을 하나도 만들지 않은 상태예요. 교사 패널 → 반/모둠 이름 관리에서 먼저 하나 이상 추가해 주세요.
유튜브 음악이 재생되지 않아요.
① 학교 네트워크에서 유튜브가 차단된 경우 → 파일 업로드 방식 사용. ② 영상의 퍼가기가 차단된 경우 → 다른 영상(클래식·국악)으로 교체하세요.
AI가 추천한 시/글감/기사가 이상해요.
입력이 너무 짧거나 모호할 때 발생해요. 구체적으로 입력하면 더 좋은 결과가 나와요. 다시 시도 버튼을 눌러보세요.
학생 활동 기록이 다른 기기에서 안 보여요.
각 앱의 학생 활동(진행 단계·작성 글자 수 등)은 그 학생이 접속한 기기·브라우저에만 저장돼요. 교사는 그 기기의 교사 패널에서 CSV로 내보내 통합 관리하세요. (반/모둠 이름 목록과 "신문으로 보내기"한 글은 Firebase로 연결되어 있어 기기 간 공유가 돼요 — 활동 기록과는 별개예요)
TTS 소리가 나오지 않아요.
Chrome 브라우저 사용 여부를 확인하세요. 첫 사용자 클릭 이후에만 소리가 활성화되는 경우가 있어요. 기기 볼륨·음소거도 확인해 주세요.
AI 수정 기능을 쓰면 원래 글이 사라지나요?
아니요! AI 수정 결과가 별도 박스로 표시되고, 적용하기 버튼을 눌러야만 바뀌어요. 취소 버튼을 누르면 원래 글이 유지됩니다.
Gemini API 사용 비용이 발생하나요?
Google AI Studio 무료 티어는 분당 15회, 하루 1,500회 요청이 무료예요. 일반 수업 활용 범위에서는 비용이 발생하지 않습니다.
시/글쓰기 앱에서 "신문으로 보내기"를 눌렀는데 신문 앱 가져오기 목록에 안 보여요.
두 앱에서 같은 이름으로 로그인했는지 확인하세요. 가져오기 목록은 로그인한 이름이 똑같은 글만 보여줘요. 신문 앱에서 "🔄 새로고침"도 한번 눌러보세요.
AI 삽화 생성이 안 돼요.
신문 앱의 AI 삽화는 gemini-2.5-flash-image 모델을 따로 호출해요. 같은 API 키를 쓰지만 이미지 생성 권한이 막혀있는 키라면 실패할 수 있어요. Google AI Studio에서 발급한 일반 키라면 보통 문제없이 작동해요.
